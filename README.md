# 웹사이트 배포 가이드 (GitHub Pages + QR)

## 1. 사진 넣기
`images/` 폴더에 아래 이름(jpg)으로 사진을 저장하세요:

| 파일명 | 사진 |
|---|---|
| `images/profile.jpg` | 메인 프로필 사진 |
| `images/field-1.jpg` | 방글라데시 필드 사진 1 |
| `images/field-2.jpg` | 방글라데시 필드 사진 2 |
| `images/startup-1.jpg` | 스타트업 우승 사진 1 |
| `images/startup-2.jpg` | 스타트업 우승 사진 2 |

사진이 없는 자리는 자동으로 숨겨지므로 일부만 넣어도 괜찮아요.
팁: 웹용으로 가로 1200px 이하, 500KB 이하로 줄이면 로딩이 빨라요.

## 2. GitHub Pages에 올리기
1. github.com 가입/로그인
2. 새 저장소 만들기 → 이름을 정확히 `<내아이디>.github.io` 로 (예: `jiyeongpyo.github.io`)
3. "uploading an existing file" 클릭 → 이 폴더의 모든 파일(index.html, CV PDF, images 폴더 안 사진들)을 드래그해서 업로드 → Commit
4. 1~2분 후 `https://<내아이디>.github.io` 에서 확인

## 3. QR 코드 만들기
사이트 주소가 생기면 저(Claude)한테 주소를 알려주세요 — QR 이미지를 바로 만들어드릴게요.
직접 하려면: qr-code-generator.com 등에 URL 입력 → PNG 다운로드.

## 4. 수정하고 싶을 때
`index.html`을 텍스트 편집기로 열어 문구를 고친 뒤 GitHub에 다시 업로드하면 됩니다. 저에게 다시 요청하셔도 돼요.
